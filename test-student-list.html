Student List View Test

Frontend: http://localhost:3000

Backend API: http://localhost:3001/api/students

Test Steps:

  1. Open the frontend at http://localhost:3000
  2. Login with development credentials
  3. Navigate to "Alunos" (Students) page
  4. Look for the view toggle buttons (cards/list icons) in the top right
  5. Click the list view button (lines icon) to switch to list view
  6. Verify the table layout shows: Student, Level, Age, Last Evaluation, Actions
  7. Click the cards view button (grid icon) to switch back to card view
  8. Test search functionality in both views
  9. Test level filtering in both views

Expected Features:

  • ✅ View mode toggle (cards/list)
  • ✅ List view with table layout
  • ✅ Profile images or initials in list view
  • ✅ Level badges with colors
  • ✅ Age calculation
  • ✅ Last evaluation date formatting
  • ✅ Action buttons (view, edit, delete)
  • ✅ Search functionality
  • ✅ Level filtering
  • ✅ Pagination (if more than 20 students)
